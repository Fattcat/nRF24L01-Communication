# nRF24L01 - Communication using Serial Monitor
nRF24L01-Communication between 2 Arduino UNO Devices with custom messages.

# Connection
Arduino nRF24L01 --> Arduino UNO
- VCC on the nRF24L01 to 3.3V
- GND - GND
- CE - D9
- CSN - D10
- SCK - D13
- MOSI - D11
- MISO - D12
