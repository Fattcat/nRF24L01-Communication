# nRF24L01 - Communication
nRF24L01-Communication devices with custom messages and channels.

# Connection
Arduino nRF24L01 --> Arduino UNO
- VCC on the nRF24L01 to 3.3V
- GND - GND
- CE - D9
- CSN - D10
- SCK - D13
- MOSI - D11
- MISO - D12
 
- Default Freq -> 2476 MHz
- Default Channel -> 76

# Librarry Installation
- RF24.h
- SPI.h
